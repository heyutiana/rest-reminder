# 休息提醒助手

桌面休息提醒软件，定时提醒你休息，保护眼睛和身体。集成番茄钟、Todo 任务、AI 专注助手、悬浮球、数据统计。

## 功能

### 计时与休息
- 自定义工作时长（1-120 分钟）、休息时长（1-30 分钟）、长休息（5-60 分钟）
- 每 N 个周期长休息（2-10）
- 4 个快捷预设（番茄钟 / 深度专注 / 学习模式 / 短冲刺）
- 开始 / 暂停 / 继续 / 跳过 / 重置
- 挂钟驱动计时：休眠/合盖后恢复立即同步真实时间，不漂移
- 稍后提醒时长可自定义（1-30 分钟）
- 提示音（WebAudio 三音和弦，可开关）
- 桌面通知（工作结束 / 休息结束 / 稍后提醒）

### Todo 任务
- 添加 / 勾选 / 编辑 / 删除任务
- 父子任务（手动或 AI 拆解生成）
- 设为当前专注任务（仅叶子任务）
- 专注次数记录与统计
- 优先级标签（高/中/低）
- 清空已完成 / 清空全部
- 自绘确认弹窗（不依赖系统原生 confirm）

### 数据统计
- 今日：专注分钟、完成周期、休息次数、Todo 完成率
- 每日专注目标进度
- 最近 7 天专注趋势柱状图
- 今日任务专注分布
- CSV 导出（Excel 友好，含 BOM）
- 数据备份导出 / 导入（JSON）
- 清空历史统计（不影响今日）

### AI 专注助手
- 目标拆解 → 生成 1 父 + N 子任务（预览后确认加入 Todo）
- 今日计划 / 今日复盘 / 休息建议 / 周报
- 休息页自动生成休息建议
- 未启用 AI 时休息建议回退到本地建议池
- OpenAI 兼容协议（默认 DeepSeek），API Key 加密存储
- 请求竞态保护、按钮 busy 锁、详细中文错误提示

### 系统集成
- 系统托盘（图标 + 菜单 + 双击恢复 + 实时状态）
- 关闭窗口 = 最小化到托盘
- 开机自启动
- 全局热键（Ctrl+Shift+P 开始/暂停 · Ctrl+Shift+S 跳过，可开关）
- 无边框自绘标题栏（最小化 / 迷你模式 / 托盘 / 关闭）

### 悬浮球（迷你模式）
- 72×72 始终置顶、透明、跳过任务栏、全工作区可见（含全屏）
- "药瓶注水"视觉，7 种状态配色
- 点击恢复主窗口、右键菜单、自定义拖拽
- 跟随主窗口所在显示器
- 实时同步剩余时间（主窗口隐藏后仍能自算）

### 个性化
- 主题切换（深色 / 浅色 / 跟随系统）
- 窗口可调整大小、位置记忆（多显示器可见性校验）

## 安装

```bash

```

```bash

```

```bash

npm run build      # NSIS 安装包
```

## 使用

1. 点击「开始」启动工作计时
2. 时间到后弹出休息提醒，可选「开始休息」「稍后提醒」「跳过」
3. 在 Todo 页管理任务，设为当前专注任务
4. 在统计页查看进度，导出 CSV 或备份
5. 在设置页自定义时间参数、主题、稍后提醒时长、AI 等
6. 关闭窗口会最小化到系统托盘

## 安全

- contextIsolation + preload 受控 IPC 暴露
- 两页 CSP（script-src 'self'，无 unsafe-inline/eval）
- IPC 入参校验、localStorage `__proto__` 防护
- AI API Key 用 safeStorage 加密存储
- 仅允许 HTTPS baseUrl + SSRF 防护（拦截内网/元数据端点）
