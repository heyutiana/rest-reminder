# 休息提醒助手 - 产品验收清单

## 计时流程

- [ ] 启动后显示“准备开始”，时间为设置的工作时长。
- [ ] 点击“开始”后进入“工作中”，倒计时正常递减。
- [ ] 点击“暂停”后倒计时停止，按钮变为“继续”。
- [ ] 点击“继续”后倒计时继续。
- [ ] 点击“重置”后回到准备开始状态。
- [ ] 工作自然结束后弹出休息页，并显示“开始休息 / 稍后提醒 / 跳过休息”。
- [ ] 工作自然结束后完成周期 +1。
- [ ] 工作自然结束后当前 Todo 的专注次数 +1。
- [ ] 工作中点击“跳过”后进入休息页，完成周期 +1，当前 Todo 专注次数 +1。

## 休息流程

- [ ] 休息页点击“开始休息”后休息倒计时开始。
- [ ] 休息自然结束后休息次数 +1。
- [ ] 休息自然结束后回到准备开始状态。
- [ ] 点击“稍后提醒”后回到计时页，并在 5 分钟后再次提醒。
- [ ] 点击“跳过休息”后回到准备开始状态。
- [ ] 跳过休息不增加休息次数。
- [ ] 跳过休息不弹出“休息结束”通知。

## Todo 流程

- [ ] Todo 页可以输入任务并点击“添加”。
- [ ] Todo 页可以按 Enter 添加任务。
- [ ] 空内容不能添加。
- [ ] 任务可以勾选完成，再次点击可以取消完成。
- [ ] 完成当前任务后，计时页当前任务自动变为“未选择”。
- [ ] 删除当前任务后，计时页当前任务自动变为“未选择”。
- [ ] 清空已完成后，未完成任务仍保留。
- [ ] 清空全部后，任务列表为空。
- [ ] 清空全部后可以继续添加新 Todo。
- [ ] 设置当前任务后，计时页显示当前任务名称。
- [ ] 工作周期完成后，当前任务专注次数增加。

## 统计页

- [ ] 今日专注分钟与计时页一致。
- [ ] 今日完成周期与计时页一致。
- [ ] 今日休息次数与计时页一致。
- [ ] Todo 完成数显示为“已完成 / 总数”。
- [ ] 最近 7 天柱状图能正常显示。
- [ ] 今日 Todo 完成率进度条能正常显示。
- [ ] 今日任务专注分布能正常显示。
- [ ] 删除或清空 Todo 后，今日任务专注分布仍保留已有专注记录。
- [ ] 清空历史统计不会清空今日数据。

## 设置页

- [ ] 工作时长可以调整并保存。
- [ ] 休息时长可以调整并保存。
- [ ] 长休息时长可以调整并保存。
- [ ] 每几个周期长休息可以调整并保存。
- [ ] 提示音开关可以保存。
- [ ] 重置今日统计后专注分钟、完成周期、休息次数归零。
- [ ] 重置今日统计后周期圆点归零。

## 托盘和窗口

- [ ] 点击最小化按钮后窗口最小化到任务栏。
- [ ] 点击托盘按钮后窗口隐藏到系统托盘。
- [ ] 点击关闭按钮后窗口隐藏到系统托盘，不退出程序。
- [ ] 双击托盘图标可以恢复窗口。
- [ ] 托盘菜单“显示窗口”可以恢复窗口。
- [ ] 托盘菜单“暂停”可以暂停工作计时。
- [ ] 托盘菜单“继续”可以恢复暂停的计时。
- [ ] 托盘菜单“跳过当前”可以跳过当前工作或休息阶段。
- [ ] 托盘菜单“退出”可以真正退出程序。

## 跨天和历史

- [ ] 软件跨天运行时会归档前一天数据。
- [ ] 软件跨天运行时今日统计归零。
- [ ] 软件跨天运行时今日 Todo 清空。
- [ ] 软件跨天运行时周期圆点归零。
- [ ] 第二天打开软件时，上一天数据仍能在最近 7 天趋势中体现。

## 安装包

- [ ] 安装包可以启动安装流程。
- [ ] 安装时可以选择安装目录。
- [ ] 安装后桌面快捷方式存在。
- [ ] 安装后开始菜单快捷方式存在。
- [ ] 通过快捷方式可以启动软件。
- [ ] Windows 卸载入口可以卸载软件。
